머신러닝(Machine Learning)은 데이터로부터 패턴을 학습하는 인공지능의 한 분야입니다.

## 다루는 주제

- 지도학습: 회귀, 분류
- 비지도학습: 군집화, 차원축소
- 신경망과 딥러닝
- 모델 평가와 검증
- 최적화 기법

## 연결되는 분야

- **선형대수**: 행렬 연산, 고유값 분해
- **확률/통계**: 확률적 모델, 베이지안 추론
- **미적분**: 경사하강법, 역전파
