시스템(Systems)은 컴퓨터 하드웨어와 소프트웨어의 동작 원리를 다룹니다.

## 다루는 주제

- 운영체제: 프로세스, 메모리, 파일 시스템
- 컴퓨터 구조: CPU, 메모리 계층
- 네트워크: TCP/IP, HTTP
- 데이터베이스: RDBMS, NoSQL
- 분산 시스템

## 연결되는 분야

- **알고리즘**: 스케줄링, 캐싱
- **보안**: 시스템 보안, 암호학
- **클라우드**: 가상화, 컨테이너
